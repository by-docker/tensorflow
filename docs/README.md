# tensorflow

## Build docker container
> make container

## Build docker container with Jupyter
> make jupyter

## Run docker container with Jupyter
```bash
make run_notebooks NOTEBOOKS_DIR=$ABSOLUTE_PATH
```
## Build GPU-enabled docker container
> make gpu_container

## Build GPU-enabled docker container with Jupyter
> make gpu_jupyter

## Run GPU-enabled docker container
```bash
make run_gpu_container
```
